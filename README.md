# 🎓 Student Manager — FastAPI Project

A full-stack Student Management System built with **FastAPI** (backend) and a custom **HTML/CSS/JS** frontend.

---

## 📁 Project Structure

```
student-manager/
├── main.py            ← FastAPI backend (all API routes)
├── requirements.txt   ← Python dependencies
├── students.json      ← Data storage (auto-created on first use)
└── static/
    └── index.html     ← Frontend UI
```

---

## ⚙️ Setup & Run

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Start the server
```bash
uvicorn main:app --reload
```

### 3. Open in browser
```
http://localhost:8000
```

---

## 🚀 Features

- ➕ **Add** students with Name, ID, Standard, Marks, Description
- ✏️ **Edit** any student record inline
- 🗑️ **Delete** students from the table or by ID
- 🔍 **Search** students by name, ID or standard
- 📊 **Live stats** — total students, average marks, top score
- 💾 Data persists in `students.json`

---

## 📡 API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/students` | Get all students |
| GET | `/students/{id}` | Get one student |
| POST | `/students/{id}` | Add a student |
| PUT | `/students/{id}` | Update a student |
| DELETE | `/students/{id}` | Delete a student |

### API Docs (auto-generated by FastAPI)
```
http://localhost:8000/docs
```
